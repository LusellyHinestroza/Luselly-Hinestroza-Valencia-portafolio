Proyectos

Desarrollé una aplicación web para la gestión de tareas que permite a los usuarios agregar, marcar y eliminar tareas de manera sencilla. La plataforma incluye funcionalidades como marcar tareas como completadas o urgentes, así como eliminarlas de forma individual. Además, cuenta con un botón para eliminar todas las tareas completadas y un sistema de contadores que muestra en tiempo real el total de tareas, tareas urgentes, tareas completadas y tareas eliminadas. El desarrollo de este proyecto se realizó utilizando tecnologías como JavaScript, HTML y CSS, enfocándome en crear una interfaz clara, funcional y fácil de usar. Asimismo, utilicé Git y GitHub para el control de versiones y la publicación del proyecto.
